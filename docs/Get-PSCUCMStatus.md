# Get-PSCUCMStatus

## SYNOPSIS
Get the status of the current CUCM Connection

## SYNTAX

```
Get-PSCUCMStatus [<CommonParameters>]
```

## DESCRIPTION
Get the status of the current CUCM Connection.
Does *not* return the credential.

## EXAMPLES

### EXAMPLE 1
```
Get-PSCUCMStatus
```
