# Find-PSCUCMLineGroup

## SYNOPSIS
Find Line Groups

## SYNTAX

```
Find-PSCUCMLineGroup [[-name] <String>] [-EnableException] [<CommonParameters>]
```

## DESCRIPTION
Find Line Groups within CUCM Environment

## EXAMPLES

### EXAMPLE 1
```
Find-PSCUCMLineGroup %
```

Find all Line Groups
